# Core Development Agents

Foundational agents covering the full development lifecycle — backend services, frontend interfaces, APIs, mobile, desktop, and real-time systems.

## Available Agents (11)

| Agent | Role |
|-------|------|

| [**backend-developer**](backend-developer.md) | Server-side expert for scalable APIs |
| [**design-bridge**](design-bridge.md) | Design-to-code translator |
| [**electron-pro**](electron-pro.md) | Desktop application expert (Electron) |
| [**frontend-developer**](frontend-developer.md) | UI/UX specialist for React, Vue, Angular |

## Install

**GitHub Copilot** — copy `.agent.md` files to `.github/agents/` in your project.

**OpenCode** — copy `.md` files to `.opencode/agents/` in your project.

Use the `install-agents.sh` script at the repo root for automated installation.
