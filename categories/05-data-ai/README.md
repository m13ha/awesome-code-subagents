# Data & AI Agents

Agents for data engineering, database work, and AI/ML development.

## Available Agents (3)

| Agent | Role |
|-------|------|
| [**postgres-pro**](postgres-pro.md) | PostgreSQL expert — schema, queries, performance |
| [**prompt-engineer**](prompt-engineer.md) | LLM prompt design and optimization |
| [**reinforcement-learning-engineer**](reinforcement-learning-engineer.md) | RL systems and agent training |

## Quick Selection Guide

| Need to... | Agent |
|------------|-------|
| Optimise SQL / schema design | **postgres-pro** |
| Craft or tune LLM prompts | **prompt-engineer** |
| Build RL agents or environments | **reinforcement-learning-engineer** |

## Install

**GitHub Copilot** — copy `.agent.md` files to `.github/agents/` in your project.

**OpenCode** — copy `.md` files to `.opencode/agents/` in your project.

Use the `install-agents.sh` script at the repo root for automated installation.
