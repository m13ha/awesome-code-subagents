# Developer Experience Agents

Agents that improve the development workflow — tooling, documentation, and code health.

## Available Agents (4)

| Agent | Role |
|-------|------|
| [**cli-developer**](cli-developer.md) | CLI tool design and development |
| [**documentation-engineer**](documentation-engineer.md) | Technical documentation specialist |
| [**readme-generator**](readme-generator.md) | README and project docs writer |
| [**refactoring-specialist**](refactoring-specialist.md) | Code refactoring and modernisation |

## Quick Selection Guide

| Need to... | Agent |
|------------|-------|
| Build a CLI tool | **cli-developer** |
| Write technical documentation | **documentation-engineer** |
| Generate a README | **readme-generator** |
| Refactor or clean up code | **refactoring-specialist** |

## Install

**GitHub Copilot** — copy `.agent.md` files to `.github/agents/` in your project.

**OpenCode** — copy `.md` files to `.opencode/agents/` in your project.

Use the `install-agents.sh` script at the repo root for automated installation.
