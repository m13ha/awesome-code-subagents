# Infrastructure Agents

Agents specializing in cloud, DevOps, networking, and platform engineering.

## Available Agents (6)

| Agent | Role |
|-------|------|
| [**cloud-architect**](cloud-architect.md) | Cloud architecture and design |
| [**deployment-engineer**](deployment-engineer.md) | CI/CD and release automation |
| [**devops-engineer**](devops-engineer.md) | DevOps practices and tooling |
| [**network-engineer**](network-engineer.md) | Networking and connectivity |
| [**platform-engineer**](platform-engineer.md) | Internal developer platform builder |
| [**security-engineer**](security-engineer.md) | Infrastructure security specialist |

## Quick Selection Guide

| Need to... | Agent |
|------------|-------|
| Design cloud architecture | **cloud-architect** |
| Set up CI/CD pipelines | **deployment-engineer** |
| Automate ops workflows | **devops-engineer** |
| Configure networks | **network-engineer** |
| Build internal platforms | **platform-engineer** |
| Harden infrastructure security | **security-engineer** |

## Install

**GitHub Copilot** — copy `.agent.md` files to `.github/agents/` in your project.

**OpenCode** — copy `.md` files to `.opencode/agents/` in your project.

Use the `install-agents.sh` script at the repo root for automated installation.
