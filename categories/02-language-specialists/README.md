# Language Specialists

Agents with deep expertise in specific programming languages and frameworks.

## Available Agents (6)

| Agent | Role |
|-------|------|
| [**expo-react-native-expert**](expo-react-native-expert.md) | Expo and React Native specialist |
| [**flutter-expert**](flutter-expert.md) | Dart/Flutter mobile and cross-platform |
| [**golang-pro**](golang-pro.md) | Go language expert |
| [**javascript-pro**](javascript-pro.md) | Vanilla JS and ecosystem specialist |
| [**react-specialist**](react-specialist.md) | React library expert |
| [**typescript-pro**](typescript-pro.md) | TypeScript type system and tooling |

## Quick Selection Guide

| Need to... | Agent |
|------------|-------|
| Build with Expo / React Native | **expo-react-native-expert** |
| Build a Flutter app | **flutter-expert** |
| Write idiomatic Go | **golang-pro** |
| Work on vanilla JS or Node | **javascript-pro** |
| Build a React application | **react-specialist** |
| Add strong typing to JS | **typescript-pro** |

## Install

**GitHub Copilot** — copy `.agent.md` files to `.github/agents/` in your project.

**OpenCode** — copy `.md` files to `.opencode/agents/` in your project.

Use the `install-agents.sh` script at the repo root for automated installation.
