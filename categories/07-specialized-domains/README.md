# Specialized Domain Agents

Agents for specific industry and technology domains.

## Available Agents (3)

| Agent | Role |
|-------|------|
| [**game-developer**](game-developer.md) | Game development and engine work |
| [**mobile-app-developer**](mobile-app-developer.md) | Native mobile app specialist |
| [**payment-integration**](payment-integration.md) | Payment gateway and fintech integration |

## Quick Selection Guide

| Need to... | Agent |
|------------|-------|
| Build a game or game mechanic | **game-developer** |
| Develop a native mobile app | **mobile-app-developer** |
| Integrate payments or fintech APIs | **payment-integration** |

## Install

**GitHub Copilot** — copy `.agent.md` files to `.github/agents/` in your project.

**OpenCode** — copy `.md` files to `.opencode/agents/` in your project.

Use the `install-agents.sh` script at the repo root for automated installation.
