# Quality & Security Agents

Agents focused on testing, security auditing, performance, and code quality.

## Available Agents (9)

| Agent | Role |
|-------|------|
| [**architect-reviewer**](architect-reviewer.md) | Architecture and design review |
| [**chaos-engineer**](chaos-engineer.md) | Chaos engineering and fault injection |
| [**code-reviewer**](code-reviewer.md) | Code review and best practices |
| [**compliance-auditor**](compliance-auditor.md) | Regulatory and standards compliance |
| [**penetration-tester**](penetration-tester.md) | Ethical hacking and pen testing |
| [**performance-engineer**](performance-engineer.md) | Performance analysis and optimization |
| [**qa-expert**](qa-expert.md) | Quality assurance and test planning |
| [**security-auditor**](security-auditor.md) | Application security auditing |
| [**test-automator**](test-automator.md) | Automated test suite development |

## Quick Selection Guide

| Need to... | Agent |
|------------|-------|
| Review system design | **architect-reviewer** |
| Test resilience under failure | **chaos-engineer** |
| Review code for quality | **code-reviewer** |
| Check regulatory compliance | **compliance-auditor** |
| Run penetration tests | **penetration-tester** |
| Analyse and improve performance | **performance-engineer** |
| Plan and coordinate QA | **qa-expert** |
| Audit application security | **security-auditor** |
| Build automated test suites | **test-automator** |

## Install

**GitHub Copilot** — copy `.agent.md` files to `.github/agents/` in your project.

**OpenCode** — copy `.md` files to `.opencode/agents/` in your project.

Use the `install-agents.sh` script at the repo root for automated installation.
